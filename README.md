OVERVIEW
A full-stack Freelancing Application developed using the MERN stack.
Clients can post jobs and freelancers can browse and apply for jobs.

TECH STACK
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

FEATURES
- User registration and login
- Role-based access (Client / Freelancer)
- Job posting and job application
- Secure REST APIs

PROJECT STRUCTURE
freelancing-application-mern/
- client/
- server/
- screenshots/
- README.md
- .gitignore

HOW TO RUN

Backend:
cd server
npm install
npm start

Frontend:
cd client
npm install
npm start
